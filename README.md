# Clone

```sh
$ git clone https://github.com/righ/dotfiles ~/dotfiles/
```

# Get environments and write it to `~/dotfiles/.envrc` 

- `GITHUB_TOKEN`: https://github.com/settings/tokens
- `NGROK_TOKEN`: https://dashboard.ngrok.com/auth

# Execution

## Once
```sh
# run as sudoer
$ ./setup.sh
```

## Each time

```sh
$ ./deploy.sh
```
